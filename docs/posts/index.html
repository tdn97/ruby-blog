Schireson Pidlist: a package for building Nielsen pidlists

2019-01-31
Building pidlists shouldn’t have to be hard. Installation schireson-pidlist requires Python 3.6+. If you want to query Postgres-derived database-backend pidlists (e.g., the tv Redshift database): pip install --index-url https://artifactory.schireson.com/artifactory/api/pypi/pypi/simple schireson-pidlist[postgres] If you want to query Athena database-backend pidlists: pip install --index-url https://artifactory.schireson.com/artifactory/api/pypi/pypi/simple schireson-pidlist[athena] Most users should use one of the above commands.
