Installation

Its recommended to install this tool globally with either pipx

$ # If you haven't installed pipx
$ curl https://raw.githubusercontent.com/cs01/pipx/master/get-pipx.py | python3

$ # Then install, using one of the below commands
$ pipx install -i https://artifactory.schireson.com/artifactory/api/pypi/pypi/simple lucha

Additionally you can set up shell completion for zsh or bash by evaluating one of the following lines in your .zshrc or .bashrc

# zsh
eval "$(_LUCHA_COMPLETE=source_zsh lucha)"

# bash
eval "$(_LUCHA_COMPLETE=source lucha)"
